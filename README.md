<p align="center">
    A content transformation system that will summarize pdf / videos and then create editable notes based on the summary generated
</p>

<br/>

## Features
Users should be able to:

- upload a pdf/videos
- resume pdf with specific filter
- Generate notes
- Create, read, update, and delete notes
- Search notes by title, tag, and content
- export notes

<br/>

## Tech stack

- React with TypeScript, Vite, Tailwind, ShadcnUI
- Express with Postgres and Sequelize
